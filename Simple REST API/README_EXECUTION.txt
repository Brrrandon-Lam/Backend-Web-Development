HOW TO START SERVER:
1) Clone Repo
2) Open terminal, navigate to saved location of the repository
3) Ensure dependencies are installed
4) 'npm start'

HOW TO USE TEST SCRIPT:
1) Open Postman
2) Import 'lamb_a1_testscript_CS493.postman_collection'
3) Run script on Postman

Note: Reset server after runs, the delete statements in the test script will destroy everything else.
Additionally you could run manual inputs.

To create a new user, do POST('/users')
-Must include atleast a username.
